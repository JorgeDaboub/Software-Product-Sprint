Undergraduate Research

Computer Science Research with Dr. Christian Poellabauer

Partnering with the University of Illinois at Urbana-Champaign we aimed to improve an already built medical application which served to aid doctors in emergency situations. The applications receive inputs in a tablet by the medical staff relating to the condition and actions that have been taken on a patient in critical condition. The application then recommends the medical team on further steps. Our work consists of training an acoustic model to recognize speech input by the medical team to be used as an interface. Once the model was trained I set up a server and its corresponding portal to receive incoming requests from the client with a .wav file to be analyzed and transcribed. On the client-side, I used the open-source project Porcupine, which allowed me to create a python script that recognizes a wake word then records the .wav file to then be sent to the server to be decoded, allowing for a practical implementation of the project.
