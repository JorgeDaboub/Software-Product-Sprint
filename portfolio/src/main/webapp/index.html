Undergraduate Research

Computer Science Research with Dr. Christian Poellabauer

Partnered with the University of Illinois at Urbana-Champaign our research aimed to improve an already built medical application which served to aid doctors when dealing with emergency situations. The applications receive user inputs in a tablet by a medical staff member relating to the condition and actions that have been taken on a patient in critical condition. The application then recommends the medical team for further steps. My work consited of training an acoustic model to recognize speech input by the medical team to be used as an interface. The model was trained using the open source software Kaldi and trained with over 25 speakers and multiple medical terms. Once the model was trained, I set up the server machine to listen for http requests. Once a request with a .wav file was received the server would call on a python script to decode the .wav file using the acoustic model and then return the result to the client. From the client-side using the open-source project Porcupine, I programmed a python script that recognizes a wake word then records the .wav file to then be sent to the server to be decoded, allowing for a practical implementation of the project.
